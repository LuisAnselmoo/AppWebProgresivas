### crear carpetas del frontend y backend

mkdir frontend
mkdir backend <- eliminado, despues lo creamos con comando de nextjs

### crear backend

npx @nestjs/cli new backend

### correr servicio de nestjs

nest start run

### instalacion de firebase  y firebase SDK

npm install firebase
npm install firebase-admin

### instalacion de axios

npm i axios

### instalacion de decoradores de Nestjs para validar reglas de lo que esperamos recibir del cliente(Frontend)

npm i --save class-validator class-transformer

### instalacion de Json Web Tocken
npm i jsonwebtoken
